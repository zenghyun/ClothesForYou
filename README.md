#  🌈 Clothes For You
<br>

## 💻 프로젝트 소개
OpenWeatherMap API를 이용하여 기온에 따른 옷차림을 추천해주는 사이트입니다.
<br>

ㆍ`HTML`

ㆍ`CSS`

ㆍ`SASS`

ㆍ`JAVASCRIPT`

ㆍ**IDE** : Visual Studio Code


## 📌주요 기능
ㆍ 3시간 간격으로 5일간의 기온과 날씨를 알려주는 기능. [보여지는 내용](#이동할 헤드(제목))
괄호 안의 링크를 쓸 때는 띄어쓰기는 -로 연결, 영어는 모두 소문자로 작성

ㆍ 현재 시간을 기준으로 가까운 시간대의 기온과 상세 날씨, 체감온도, 습도, 풍향및 풍속을 알려주는 기능

ㆍ 평균 온도를 기준으로 기온별 옷차림을 정해주는 기능 

ㆍ 주간 날씨의 습도, 최고 기온, 최저 기온 및 날씨를 알려주는 기능 

ㆍ 유튜브 API를 이용하여 **오늘 비와?**의 채널의 최근 날씨 예보를 보여주는 기능 

ㆍ 기온별 옷차림 선정 Tip을 알려주는 기능 

ㆍ 내가 입었던 옷차림을 기록할 수 있는 기능 

ㆍ 옷을 입을 때 컬러 매치를 간단하게 도와줄 수 있는 기능
